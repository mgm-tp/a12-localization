## Packages Documentation

### TS

- codemod - code mods for recent breaking changes
- localization - localization & conversion functionality

### Other

- documentation - developer documentation and TypeDoc

## Getting Started

### How to Use It

To install the latest npm package, use npm/pnpm:

```sh
npm install @com.mgmtp.a12.utils/utils-localization-react
```

### How to Build and Run

The following tasks can be executed in the repository root or in individual folders to build specific sub-projects.

#### Prerequisites

The following tools are required in order to build this repository.

| Tool   | Version |
| ------ | ------- |
| [Node] | `^24`   |
| [pnpm] | `^10.6` |

#### Install Dependencies

```sh
pnpm install
```

#### How to Build

```sh
pnpm compile
```

#### How to Test

```sh
pnpm test
```

#### Cleaning Build Artifacts

```sh
pnpm clean
```

#### Code Quality

```sh
pnpm verify
```

#### How to Run

There are no applications or showcases to execute.
